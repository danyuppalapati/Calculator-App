A responsive calculator built using HTML, CSS, and JavaScript. The application performs basic arithmetic operations and demonstrates DOM manipulation and event handling.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Percentage
- Decimal numbers
- Clear and Delete functions
- Keyboard support
- Responsive layout

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## How to Run
1. Download or clone the repository.
2. Open `index.html` in a web browser.
